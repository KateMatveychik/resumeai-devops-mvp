# ResumeAI DevOps MVP

Командный проект по дисциплине «Технологии системного программирования».

## О проекте
ResumeAI — это AI-помощник для адаптации резюме под конкретную вакансию и генерации сопроводительного письма.

## Цель лабораторной работы №4
Настроить процессы командной разработки, сборки и подготовки MVP к демонстрации:
- инициализация репозитория;
- стандартизация работы с Git;
- контейнеризация приложения;
- настройка CI/CD;
- подготовка MVP к презентации.

## Основной функционал MVP
- загрузка резюме в формате PDF или DOCX;
- ввод текста вакансии;
- генерация адаптированного резюме;
- генерация сопроводительного письма;
- скачивание результата.

## Технологический стек
- Backend: Python, FastAPI
- Frontend: React
- База данных: PostgreSQL
- Контейнеризация: Docker, Docker Compose
- CI/CD: GitHub Actions / GitLab CI
- LLM: OpenAI API

## Структура репозитория
- `backend/` — серверная часть
- `frontend/` — клиентская часть
- `docker/` — конфигурация контейнеров
- `docs/` — документация проекта

## Правила командной работы
- основная ветка: `main`
- рабочая ветка разработки: `develop`
- новые задачи выполняются в ветках `feature/*`
- прямые push в `main` запрещены
- изменения вносятся через Pull Request

## Запуск проекта
Инструкция по запуску будет добавлена после настройки Docker Compose и CI/CD.
